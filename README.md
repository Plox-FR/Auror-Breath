# 🌬️ AUROR Breath

> Take a moment to center yourself.

**AUROR Breath** is a guided breathing application built around *heart coherence* and other relaxation techniques. With animated visuals, ambient sounds and customizable sessions, it helps you slow down, refocus and manage stress — in just a few minutes.

Bilingual interface (🇬🇧 English / 🇫🇷 French, defaults to French). Built in Python with Tkinter.

---

## ✨ Features

- **15+ breathing exercises**, from heart coherence to box breathing and the 4-7-8 sedative method
- **Animated guidance** — a smooth visual cue follows each phase (inhale, hold, exhale, hold)
- **Ambient soundscapes** — birds, sea, rain or night
- **Flexible sessions** — 5 min, 2 min, or free mode (run as long as you like)
- **Adjustable interface size** — Large, Medium, Small or Mini (handy as a discreet corner widget)
- **Bilingual** — English / French

## 🚀 Getting started

### Option 1 — Run the executable (Windows, no installation)

1. Go to the [**Releases**](../../releases) page
2. Download `auror_breath.exe`
3. Double-click to run — no installation needed

> ⚠️ **On first launch, Windows may show a SmartScreen warning** ("Windows protected your PC"). This is expected: the app isn't code-signed (the certificate costs ~150 €/year, and this is a free project). To run it anyway, click **More info → Run anyway**. The full source code is in this repo if you'd like to review or build it yourself.

### Option 2 — Run from source (any OS with Python)

```bash
git clone https://github.com/Plox-FR/Auror-Breath.git
cd Auror-Breath
python auror_breath.py
```

Requires **Python 3** with Tkinter (included in most Python installs). Note that some sound features rely on `winsound` and are Windows-specific.
